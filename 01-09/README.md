### **EDA(탐색적 데이터 분석)**
* 차트만으로도 데이터의 분포나 상관관계를 확인 할 수 있다. 
* 미래 데이터 예측 시 변수 설정
**결측치 시각화**
**피어슨 상관계수**
* 두 변수의 선형 상관 관계를 계량화한 수치입니다.
* 결과값은 -1 ~ 1 사이의 값이며,
* 양의 상관 관계가 있을수록 1에 가깝고, 음의 상관 관계가 있을수록 -1에 가깝습니다.
* 또한, 상관 관계가 없을수록 0에 가깝습니다.
